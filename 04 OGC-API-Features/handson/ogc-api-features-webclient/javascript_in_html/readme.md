# Webclients in de browser demo

- Het tonen van OGC-API-Features in een webclient kan met een Leaflet javascript, ingebakken in een html pagina. Dit is verwerkt in een opdracht die onderdeel is van de introductiecursus en staat [hier](https://github.com/Geonovum/ogc-api-workshops/blob/main/01%20introductie/handson/05-OGC-api-features-in-web-map-clients.md).

- Er is ook een opdracht die in plaats van Leaflet [MapLibre](https://github.com/Geonovum/ogc-api-workshops/blob/main/01%20introductie/handson/04-vector-tiles-in-web-map-clients.md) gebruikt. 

- index.html gebruikt de NAP Peilmerken uit de NodeJS OGC API Features Server van de opdracht [bouw_je_eigen_server](https://github.com/Geonovum/ogc-api-workshops/tree/main/04%20OGC-API-Features/handson/bouw_je_eigen_server). Zorg dat je de server uit [stap9](https://github.com/Geonovum/ogc-api-workshops/tree/main/04%20OGC-API-Features/handson/bouw_je_eigen_server/stap9) hebt draaien om deze kaart te zien.
