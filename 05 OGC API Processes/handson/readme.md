# Handson oefeningen voor OGC API Processes

## Pygeoapi

Dit is een oefening met een geconfigureerde Pygeoapi server met voorbeeld OGC API Processes. En er is een jupyter notebook client voorbeeld om de OGC API Processes aan te roepen.

## nodejs

In de https://github.com/Geonovum/ogc-api-kennissessie repo staat een demo implementatie van OGC API Processes in NodeJS.
Deze implementatie bevat voorbeelden van processen die op verschillende manieren geiplementeerd zijn:
- batch/bash script
- javascript
- docker
